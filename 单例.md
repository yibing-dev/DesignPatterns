# 单例模式

### 应用场景

* 只需要一个实例存在，比如各种的manager类和各种的factory类

##### 代码示例

``` java
/**
 * @Author yibing
 * @CreateTime 2020年6月1日 上午7:44:22
 * @Description 类描述:
 * 饿汉式
 * 类加载到内存后，就会实例化一个单例，JVM保证线程安全
 * 简单实用，推荐使用
 * 唯一缺点：不管用到与否，类装载的时候都会被实例化
 */
public class Mgr01 {
    private static final Mgr01 INSTANCE = new Mgr01();
    // 禁止new
    private Mgr01() {};
    // 初始化方法
    public static Mgr01 getInstance() {
        return INSTANCE;
    }
}
```
